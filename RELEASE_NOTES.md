# AAAntylogout 1.0.0

First production release.

## Highlights

- PvP combat tagging after player hits and projectile hits.
- 20-second combat timer by default.
- Bossbar, actionbar and chat reminders.
- Combat logout punishment with death, item drops and broadcast.
- Command blacklist/whitelist support.
- Blocked combat items and teleport causes.
- Optional WorldGuard support plus built-in local regions.
- Admin battle history and statistics.
- Config option to keep surviving opponents tagged after a death.

## Compatibility

- Paper 1.21+
- Java 21
- WorldGuard optional
