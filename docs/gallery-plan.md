# Modrinth Gallery Plan

Use clean 16:9 screenshots, preferably 1920x1080. Keep chat readable, use a simple arena, and avoid visual clutter. Suggested shader: none or very light vanilla-friendly lighting so UI text stays readable.

## Recommended Images

1. `combat-timer-actionbar.png`
   - Two players in a small PvP arena right after a hit.
   - Show bossbar at top, actionbar over the hotbar and chat start message.
   - Caption: `Combat timer with bossbar, actionbar and chat warning.`

2. `combat-logout-punishment.png`
   - One player disconnected during combat and dropped items on the ground.
   - Show broadcast message in chat.
   - Caption: `Combat logging punishment with item drops.`

3. `blocked-command.png`
   - Player tries `/spawn` during combat.
   - Show the blocked command message in chat.
   - Caption: `Configurable command restrictions during combat.`

4. `blocked-region.png`
   - Player is stopped at a spawn/protected region border while tagged.
   - Use a visible line, wall or marker blocks to show the region boundary.
   - Caption: `WorldGuard or local region protection.`

5. `admin-history.png`
   - Chat output from `/aaalo history <player>` or `/aaalo stats <player>`.
   - Caption: `Admin combat history and statistics.`

6. `config-preview.png`
   - Cropped editor screenshot showing the most important config options.
   - Include combat duration, logout punishment, command mode and visual toggles.
   - Caption: `Fully configurable behavior and messages.`

## Optional Banner/Icon Ideas

- Icon: red shield with a small broken logout arrow, dark background, no tiny text.
- Banner: two Minecraft players dueling in an arena with a clear combat timer UI overlay.
- Keep the plugin name large only on the banner, not on every screenshot.
